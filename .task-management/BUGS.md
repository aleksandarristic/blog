# BUGS

Rules:
- Use stable task IDs in format `TASK-####`.
- IDs are immutable and never reused.
- Bugs promoted to `.task-management/TODO.md` keep the same ID.
- Bugs moved to `.task-management/BACKLOG.md` keep the same ID.
- When resolving a bug, remove it from this file and append it to `.task-management/DONE.md`.
- When dropping a bug, remove it from this file and append it to `.task-management/REMOVED.md`.
